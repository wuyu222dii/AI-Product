# AI 论文共写工作台｜AI 产品经理
2026 年 xx 月 - 2026 年 xx 月

项目概述：该项目为面向本科课程论文场景的 AI 产品实践项目，旨在打造一款支持“材料解析-初稿生成-知识库证据检索-共写修改-审查申诉-导出”的智能论文共写工具，完成可演示 MVP 并接入真实 AI 能力。

1）产品规划与用户洞察：主导产品 0 到 1 规划与定义。围绕本科论文写作流程，梳理材料分散、证据不足、返工成本高、AI 生成不可信等核心痛点，拆解“上传材料 -> AI 解析 -> 材料充足性检查 -> 生成初稿 -> 共写审查 -> 导出”的主链路，明确产品定位为“AI 共写者”而非“强裁判”。

2）功能设计与 PRD 撰写：负责整体信息架构与核心功能规划，输出 PRD、页面规格、API 字段、OpenAPI 草案及数据库模型。设计材料不足拦截、AI 语义解析、图片 OCR、知识库证据片段、引用辅助、审查申诉与版本管理等机制，保证生成过程可控、可追溯、可修改。

3）技术落地与 AI 编排：推动 React + Spring Boot + Supabase PostgreSQL 的端到端实现，接入真实 AI 语义解析、初稿生成、共写改写与复审能力，完成项目知识库 MVP 与工作台内证据检索。支持从材料处理到论文导出的完整闭环，并实现可演示 Demo 跑通。

4）验证与风险治理：建立 MVP 交付与验证标准，完成后端核心路径测试和前端生产构建验证，保障基础稳定性。针对材料不足兜底、审查误判、证据链污染等风险，设计阻断、申诉和复审机制，提升 AI 产品的可信度与可用性。
