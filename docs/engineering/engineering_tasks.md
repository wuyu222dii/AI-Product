# AI 论文共写工作台 v1 前后端任务拆分清单

## 1. 开发节奏建议

按 4 个阶段推进：

1. `基础链路`
   上传、预处理、AI 解析、材料充足性检查
2. `生成链路`
   初稿生成、版本保存、共写主页面
3. `质量链路`
   审查、来源绑定、重复控制、申诉复审
4. `交付链路`
   导出、错误兜底、埋点与监控

---

## 2. 前端任务拆分

### FE-01 项目首页
- 新建项目入口
- 历史项目列表
- 项目状态展示
- 最近版本摘要

交付物：
- 首页 UI
- 项目列表请求接入
- 新建项目表单

### FE-02 上传页
- 拖拽上传
- 多文件选择
- 粘贴文本
- 链接 / DOI 输入
- 补传入口

交付物：
- 上传组件
- 文件队列状态
- 上传进度提示

### FE-03 解析状态页
- 本地预处理进度展示
- AI 语义解析状态展示
- 关键材料状态标识
- 重试入口

交付物：
- 状态页 UI
- 轮询 / websocket 状态刷新

### FE-04 解析补全页
- 展示未解析或低置信度缺口
- 手动补充文本
- 重传文件
- 原文件定位补充

交付物：
- 缺口清单组件
- 补充提交交互

### FE-05 材料不足页
- 显示无法生成说明
- 缺失项
- 建议补充内容
- 建议补充数量
- 返回上传页

交付物：
- 拦截态 UI

### FE-06 共写工作台主页面
- 中央正文编辑器
- 左侧推荐任务与审查区
- 右侧 AI 共写面板
- 下方版本记录

交付物：
- 主工作台布局
- 编辑器集成
- 版本切换

### FE-07 共写操作交互
- 重写选中内容
- 补证据
- 调结构
- 压重复
- 改表达

交付物：
- AI 操作按钮与结果回填

### FE-08 审查详情抽屉
- `仅提示 / 局部修正 / 必须确认`
- 定位到具体段落或句子
- 接受 / 忽略 / 稍后处理

交付物：
- 审查抽屉
- 操作回写

### FE-09 申诉复审页
- 提交申诉理由
- 上传补充证据
- 查看复审结果
- 带风险继续

交付物：
- 申诉表单
- 复审结果页

### FE-10 导出页
- 导出 docx
- 导出 pdf
- 导出带批注版本
- 导出状态展示

交付物：
- 导出 UI
- 导出任务状态展示

### FE-11 错误态与兜底
- AI 调用失败提示
- 部分生成提示
- 重试入口
- 回退到最后成功版本

交付物：
- 统一错误弹层 / 错误页

### FE-12 埋点与分析
- 上传成功率
- 解析失败率
- 生成失败率
- 共写操作点击率
- 导出成功率

交付物：
- 前端埋点接入

## 3. 后端任务拆分

### BE-01 项目服务
- 创建项目
- 获取项目列表
- 获取项目详情
- 删除 / 归档项目

### BE-02 文件上传服务
- 文件接收
- 对象存储落盘
- 元数据入库
- 格式校验

### BE-03 本地预处理服务
- 文本抽取
- OCR
- 页数拆分
- 图片 / 表格切片

### BE-04 AI 语义解析服务
- 材料分类
- 主题相关性分析
- 要求识别
- 论点 / 证据识别
- 语义摘要

### BE-05 Requirement Preflight 服务
- 提取老师要求
- 冲突检查
- 最小关键字段检查
- Requirement Snapshot 版本化

### BE-06 材料充足性检查服务
- 判断是否足以生成
- 缺失项归因
- 建议补充内容
- 建议补充数量

### BE-07 初稿生成服务
- 题目建议生成
- 全文框架生成
- 段落骨架生成
- 初稿生成
- 新版本写入

### BE-08 共写迭代服务
- 选中内容重写
- 补证据
- 调结构
- 压重复
- 改表达

### BE-09 来源绑定服务
- 论点与来源绑定
- 段落与来源映射
- 缺失来源检测

### BE-10 审查服务
- 生成 `仅提示 / 局部修正 / 必须确认`
- 目标范围定位
- 文本建议生成

### BE-11 申诉复审服务
- 接收申诉
- 调起复审
- 复审结果回写
- 降低影响等级 / 撤销

### BE-12 导出服务
- docx 导出
- pdf 导出
- 带批注版本导出

### BE-13 任务编排与状态机
- 异步任务入队
- 状态跟踪
- 重试机制
- 部分成功标记

### BE-14 监控与日志
- AI 调用日志
- 错误日志
- 性能监控
- 审计日志

---

## 4. 联调优先顺序

### 第一批联调
- 项目创建
- 上传
- 解析状态
- 材料不足拦截

### 第二批联调
- 初稿生成
- 共写工作台
- 新版本保存

### 第三批联调
- 审查结果
- 来源绑定
- 重复控制
- 申诉复审

### 第四批联调
- 导出
- 错误兜底
- 埋点与监控

---

## 5. 角色分工建议

### 前端
- 页面与组件实现
- 编辑器集成
- 状态轮询 / 实时更新
- 交互反馈

### 后端
- 上传与对象存储
- 业务 API
- AI 编排
- 数据库设计与任务系统

### AI / 算法
- 语义解析 Prompt / 工作流
- 材料充足性判断逻辑
- 初稿生成策略
- 审查与复审策略

### QA
- 文件格式兼容性
- AI 失败兜底
- 拦截态与可恢复性
- 导出一致性
